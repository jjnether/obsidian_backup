Condition Functions are a set of special functions the Creation Kit can access directly from the game with no scripting. They are usually presented as a drop-down list when adding conditions to an object, as in the example pictured below.

  
For details about setting up Condition Functions in the editor, visit the [Conditions](https://ck.uesp.net/wiki/Conditions "Conditions") page.

[![](https://ck.uesp.net/w/images/thumb/f/f3/Conditionfunctionexample.jpg/400px-Conditionfunctionexample.jpg)](https://ck.uesp.net/wiki/File:Conditionfunctionexample.jpg)

Adding a `GetIsID` condition function to a magic effect.
